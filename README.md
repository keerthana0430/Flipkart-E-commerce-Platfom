# Flipkart Online E-Commerce Database Management System

## CHAPTER-1

---

## INTRODUCTION
An Online E-Commerce Database Management System is designed to manage and organize the data involved in an online shopping platform efficiently. This project is based on Flipkart, one of India's leading e-commerce platforms, which offers a wide range of products such as electronics, fashion, home appliances, books, groceries, and more.

The system stores and manages customer information, product details, seller records, orders, payments, delivery status, and reviews using a relational database. A Database Management System (DBMS) helps ensure data accuracy, reduces redundancy, improves security, and enables fast retrieval of information.

This project demonstrates how a well-designed database supports the smooth functioning of an e-commerce platform by allowing users to browse products, place orders, make payments, and track deliveries efficiently. It also helps administrators manage inventory, customers, and sales records effectively.

---

## PROBLEM STATEMENT
Managing customer, product, order, seller, payment, and delivery data in an online e-commerce platform is a complex task when done manually or without a proper system. In traditional methods, data may be stored in separate files or handled manually, which can lead to issues like data redundancy, inconsistency, duplication, and human errors. It also becomes difficult to maintain large volumes of data and retrieve information quickly.

In an e-commerce system like Flipkart, thousands of transactions happen every second, including product browsing, order placement, payment processing, and delivery tracking. Without an efficient database system, it is not possible to handle this huge amount of data in an organized and secure manner.

Therefore, there is a need to develop a Flipkart Online E-Commerce Database Management System that can efficiently store, manage, and retrieve all necessary information in a structured way. This system ensures data accuracy, improves security, reduces redundancy, and supports smooth e-commerce operations.

---

## OBJECTIVE
The primary objective of this project is to design and develop a Database Management System for Flipkart’s online e-commerce platform that efficiently stores, manages, and retrieves all essential information related to customers, products, sellers, orders, payments, stock, and delivery details.

The system ensures structured data storage, data integrity, and security. It also reduces redundancy using relational database design with primary and foreign keys.

This project demonstrates DBMS concepts such as relational design, normalization, SQL queries, and transaction management.
